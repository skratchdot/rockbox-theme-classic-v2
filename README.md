classic-v2 (A Rockbox Theme for the Sansa Clip+)
------------------------------------------------

### Installation Instructions: ###
1) Install Rockbox on your Sansa Clip+ by downloading
   from here: [http://www.rockbox.org/download/](Rockbox).  
2) Now you can install the "classic-v2" theme by downloading the .zip
   from here: [https://github.com/skratchdot/rockbox-theme-classic-v2](classic-v2).
